반복 거래 자동 생성
입출금 내역에서 반복 버튼을 눌러 반복 거래로 지정하면, 매월 같은 날짜로 예정 항목을 자동 생성합니다.
사용 방법
1. 입출금 내역 탭에서 반복적으로 발생하는 거래 행의 반복 버튼 클릭
2. 반복 거래로 지정된 항목이 아래 목록에 자동으로 추가됩니다
3. 로딩 시 자동으로, 또는 지금 생성 버튼으로 익월~ 예정 항목이 입출금 내역에 추가됩니다
거래내용 입금액 출금액 구분 중분류 원본 거래일 마지막 생성
지정된 반복 거래가 없습니다.
입출금 내역 탭에서 각 행의 반복 버튼을 눌러 지정하세요.
변경사항은 저장 & 재매핑 버튼을 눌러야 기존 입출금 내역에 반영됩니다.
